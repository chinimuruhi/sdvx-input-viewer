SDVX Input Viewer

コナステ SOUND VOLTEX 専用コントローラ向けの入力表示ツール

使用する

概要

コナステ SOUND VOLTEX 専用コントローラ向けの入力表示ツールです。

動作のイメージは以下の動画をご覧ください。

想定用途

  • 入力の確認
  • OBS Studioで取り込んで動画投稿・配信

使用方法

Generate画面で「Generate」を押下してください。必要に応じて設定を変更することができます。

設定データが消えた時に備え、Export画面から設定をバックアップしておくことを推奨します。

OBS Studioに取り込む（ブラウザソース）
  1. 利用推奨環境のブラウザでページを開く。
  2. Generate画面に遷移して各種設定を行う。この際、一般（General）のBackGround TransparentをONにする。
  3. 「Generate」ボタンを押下する。また、表示された画面のURLをコピーする。
  4. 「ソース ＞ ブラウザ」よりブラウザソース作成したら、URL欄にコピーしたURLを貼り付け、OKを押下
OBS Studioに取り込む（ウィンドウキャプチャ）

古いバージョンのOBS Studioではブラウザソースで正常に動作しない場合があります。

  1. 利用推奨環境のブラウザで当サイトを開く。
  2. Generate画面に遷移して各種設定を行い、「Generate」ボタンを押下する。
  3. 「ソース ＞ ウィンドウキャプチャ」等でブラウザの画面を取り込む
  4. 追加したソースを右クリックして「フィルタ」を押下。
  5. 「クロップ/パッド」を追加して余計な部分を切り取り
  6. 「クロマキー」を追加して背景を削除

利用推奨環境

  • Microsoft Edge （最新版）
  • Mozilla Firefox （最新版）
  • Google Chrome （最新版）

一部機能ではLocalStorageを使用しています。無効な場合、正常に利用できない可能性があります。

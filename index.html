SDVX Input Viewer

概要

コナステ SOUND VOLTEX 専用コントローラ向けの入力表示ツールです。※未完成です。

用途

以下のような用途を想定しています。

  • 入力の確認
  • OBSで取り込んで動画投稿・配信

使用方法

Generate画面にて表示設定を行い、「Generate」を押下してください。

設定データが消えた時に備え、Export画面から設定をバックアップしておくことを推奨します。
Import画面からリストアが可能です。

利用推奨環境

JavaScriptが利用可能である、以下の環境からの利用を推奨します。

  • Microsoft Edge （最新版）
  • Mozilla Firefox （最新版）
  • Google Chrome （最新版）

（環境は一例となります）

また、設定の保存のためLocalStorageを使用しています。
ブラウザのLocalStorageが使用できない状態の場合、正常にご利用いただくことができません。

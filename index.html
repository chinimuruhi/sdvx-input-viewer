SDVX Input Viewer

概要

コナステ SOUND VOLTEX 専用コントローラ向けの入力表示ツールです。

動作のイメージは以下の動画をご覧ください。

用途

以下のような用途を想定しています。

  • 入力の確認
  • OBS Studioで取り込んで動画投稿・配信

使用方法

Generate画面で「Generate」を押下してください。必要に応じて設定を変更することができます。

設定データが消えた時に備え、Export画面から設定をバックアップしておくことを推奨します。
Import画面からリストアが可能です。

OBS Studioに取り込む場合

「ソース ＞ ブラウザ」で取り込むと正常に動作しません。（OBS Studio Ver.29.1.3で確認）

以下の手順で取り込むことを推奨します。OBSの詳細な使用方法は割愛します。

  1. 利用推奨環境のブラウザでページを開く。
  2. 「ソース ＞ ウィンドウキャプチャ」等でブラウザ画面を取り込む
  3. 追加したソースを右クリックして「フィルタ」を押下。
  4. 「クロップ/パッド」を追加して余計な部分を切り取り
  5. 「クロマキー」を追加して背景を削除

GamePad Viewerと同様に使えることが理想だったのですが、現在のOBSのブラウザではSDVXコントローラのつまみ入力が正常に取得できません...
OBSの今後のアップデート次第では同様に使えるようになるかも

利用推奨環境

JavaScriptが利用可能である、以下の環境からの利用を推奨します。

  • Microsoft Edge （最新版）
  • Mozilla Firefox （最新版）
  • Google Chrome （最新版）

（環境は一例となります）

また、一部機能でLocalStorageを使用しています。
ブラウザのLocalStorageが使用できない状態の場合、正常にご利用いただくことができない可能性がございます。
